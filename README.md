# SwiftI2V — Project Page

This repository hosts the project page for:

> **SwiftI2V: Efficient High-Resolution Image-to-Video Generation via Conditional Segment-wise Generation**

🌐 **Live page:** https://lazysheeeeeep.github.io/SwiftI2V/

SwiftI2V is a **2K image-to-video generation framework** that is **202× faster**
than end-to-end baselines while running on a single consumer GPU (RTX 4090, 24 GB).
Please visit the live page above for videos, qualitative comparisons, and ablations.
